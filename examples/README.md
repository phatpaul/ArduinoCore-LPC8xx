# examples/

Arduino-style example sketches. The classic `Blink/Blink.ino` is the
smoke test once the core implementation lands. Add more examples here
as additional API surface (SPI, Serial, etc.) becomes real.

**SKELETON.** Examples won't actually run until the core stops being
no-op stubs — tracked at
[FastLED/fbuild#479](https://github.com/FastLED/fbuild/issues/479).
