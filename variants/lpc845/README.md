# variants/lpc845

LPC845 variant. Reference board: **LPC845-BRK**.

- MCU: LPC845M301JBD64 (Cortex-M0+ @ up to 30 MHz)
- Flash: 64 KB
- SRAM: 16 KB
- USART0 / SPI0 routed via switch matrix to the LPC845-BRK Arduino-shape
  header pins.

**SKELETON.** Pin map and clock init are TBD. See `variant.cpp`.
