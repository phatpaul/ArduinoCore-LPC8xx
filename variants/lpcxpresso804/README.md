# variants/lpcxpresso804

Board variant for the NXP LPCXpresso804 board.

- FQBN: `zackees:lpc8xx:lpcxpresso804`
- MCU: LPC804M101JDH24, 32 KB flash, 4 KB SRAM
- Default upload path: CMSIS-DAP/SWD through OpenOCD
- Board model: 3.3 V Arduino-header-compatible evaluation board

The current pin table exposes direct GPIO indexes plus capability metadata.
Arduino-header labels, Grove/I2C sharing, and jumper-specific notes need a
hardware-manual audit before the map is considered final.
